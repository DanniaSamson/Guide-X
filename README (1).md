# Project Name
SkyServe Kerala: AI-Powered Last-Mile Food Delivery Optimization Platform

## Problem Statement
Urban areas in Kerala face significant challenges in last-mile food delivery due to heavy traffic congestion, narrow roads, unpredictable weather conditions, and inefficient routing systems. These factors lead to delayed deliveries, increased fuel consumption, and poor customer experience.

There is a need for an intelligent, fast, and reliable delivery system that can optimize last-mile logistics while ensuring customer data privacy and operational efficiency

## Project Description
SkyServe Kerala is an AI-powered drone-assisted food delivery platform designed to optimize last-mile delivery in urban Kerala.

The platform uses smart routing algorithms and real-time traffic analysis to determine the fastest and safest delivery paths. By integrating drone delivery, the system bypasses road congestion and significantly reduces delivery time.

How it works:
User visits the website and places a food order
The system requests location permission securely
AI analyzes:
traffic conditions
weather data
delivery distance
The system selects:
drone delivery (for faster routes) OR
alternative delivery (if unsafe)
User selects preferred delivery time
Drone is dispatched with optimized route
User tracks delivery live with ETA and real-time updates
Key Features:
AI-based route optimization
Drone-assisted delivery system
Real-time delivery tracking
ETA prediction
Traffic-aware routing
Weather-based decision making
Secure location handling
User-friendly modern interface
What makes it useful:
Faster delivery in congested urban areas
Reduced fuel usage and emissions
Improved customer satisfaction
Scalable smart logistics system
Supports future smart city infrastructure

### Tools / Models Used
Google Maps Platform APIs (Routing, Distance Matrix)
Google Gemini API (AI reasoning and optimization)
Firebase (Authentication, Database, Hosting)
Google Cloud AI tools (for data processing and predictions)

### How Google AI Was Used
Google AI is integrated into the core functionality of the platform to enable intelligent decision-making and automation.

Route Optimization:
AI models analyze traffic density, distance, and environmental factors to determine the fastest delivery route.
ETA Prediction:
Machine learning models estimate delivery time based on historical and real-time data.
Traffic Analysis:
AI processes live traffic data to identify congested routes and suggest alternative aerial paths.
Decision Engine:
Gemini AI helps decide whether to use drone delivery or alternative delivery methods based on safety and efficiency.
User Interaction:
AI enhances user experience by providing smart suggestions and dynamic delivery updates.
